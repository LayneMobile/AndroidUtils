# AndroidUtils
Helpers for Android

Download:
```groovy
repositories {
    maven { url 'http://dl.bintray.com/laynemobile/maven' }
}

dependencies {
    // Base Utils
    compile 'com.laynemobile:androidutils:0.1.3'

    // Android Drawer Utils
    compile 'com.laynemobile:androidutils-drawer:0.1.3'
}
```
